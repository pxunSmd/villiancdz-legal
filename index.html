Пользовательское соглашение (Публичная оферта)
Настоящее Пользовательское соглашение является публичной офертой в соответствии со статьёй 437 Гражданского кодекса Российской Федерации. Начало использования Сервиса означает полное и безоговорочное принятие условий настоящего Соглашения (акцепт оферты) в соответствии со статьёй 438 ГК РФ.
1 Общие положения и определения
1.1. Стороны соглашения

Исполнитель — физическое лицо Администрация сервиса VillianCDZ, предоставляющее доступ к Сервису на условиях настоящего Соглашения.

Пользователь — физическое лицо, акцептовавшее настоящую оферту путём начала использования Сервиса.

1.2. Основные определения
  1. Сервис — программный комплекс VillianCDZ, функционирующий в виде Telegram-бота, предоставляющий услуги по генерации текстового контента с использованием технологий искусственного интеллекта.
  2. Услуга генерации — предоставление Пользователю текстового контента, сформированного автоматически с использованием ИИ-алгоритмов, без гарантии его достоверности, корректности или применимости.
  3. МЭШ — Московская электронная школа, внешний сервис, не аффилированный с Исполнителем.
  4. Токен МЭШ — временный авторизационный ключ для доступа к данным МЭШ, предоставляемый самим Пользователем.
  5. Подписка — платный тарифный план, предоставляющий расширенный доступ к функциям Сервиса.
  6. Личный кабинет — персональный раздел Пользователя в интерфейсе Telegram-бота.
1.3. Возраст Пользователя

Сервис предназначен для лиц, достигших возраста 14 лет. Лица в возрасте от 14 до 18 лет вправе использовать Сервис с согласия родителей или иных законных представителей. Начало использования Сервиса означает подтверждение соответствия возрастному требованию.

2 Предмет соглашения и характер услуг
2.1. Предмет соглашения

Исполнитель предоставляет Пользователю доступ к Сервису, обеспечивающему возможность получения автоматически сгенерированного контента с использованием технологий искусственного интеллекта, а также доступ к данным учётной записи МЭШ Пользователя через предоставленный им токен.

⚠️ ВАЖНО: Сервис предоставляет исключительно услугу по генерации текстового контента средствами искусственного интеллекта. Исполнитель не оказывает образовательных, консультационных, репетиторских или иных профессиональных услуг.
  1. Весь контент, предоставляемый Сервисом, является результатом автоматической генерации алгоритмами ИИ и не проходит проверку со стороны Исполнителя.
  2. Исполнитель не гарантирует достоверность, точность, полноту, актуальность или применимость сгенерированного контента для каких-либо целей.

  4. Правильность или неправильность сгенерированных ответов не является критерием качества оказанной услуги генерации.
  5. Пользователь самостоятельно несёт ответственность за принятое решение об использовании сгенерированного контента.
2.3. Функциональные возможности Сервиса
  1. Просмотр перечня домашних заданий из учётной записи МЭШ Пользователя по предоставленному токену.
  2. Автоматическая генерация ответов на задания с настраиваемыми параметрами (время выполнения, допустимое количество ошибок).
  3. Доступ к ИИ-ассистенту для получения сгенерированных текстовых ответов на запросы Пользователя (в рамках тарифного плана).
  4. Реферальная программа для получения дополнительных единиц доступа к Сервису.

4 Стоимость и порядок оплаты
4.1. Тарифные планы
  1. Бесплатный план — 7 (семь) единиц генерации при регистрации, без дополнительной оплаты.
  2. Базовый план — 100 (сто) рублей в месяц, включает безлимитный доступ к генерации ДЗ и ИИ-ассистенту.
  3. Актуальный перечень тарифных планов и их условия отображаются в команде /pay.
4.2. Порядок оплаты
  1. Оплата производится через платёжные системы, указанные в интерфейсе Сервиса.

  2. Для оформления возврата необходимо обратиться в службу поддержки через команду /sup с указанием реквизитов платежа.
  3. Возврат средств за частично использованный период подписки не производится, за исключением случаев технической невозможности предоставления Сервиса по вине Исполнителя.

1.1. Оператор персональных данных

Оператором персональных данных является физическое лицо Администрация сервиса VillianCDZ (далее — «Оператор»), обеспечивающее функционирование Telegram-бота VillianCDZ.

1.2. Применимое законодательство

Обработка персональных данных осуществляется в соответствии с:

  1. Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».
  2. Постановлением Правительства РФ от 01.11.2012 № 1119 «Об утверждении требований к защите персональных данных».
  3. Приказом ФСТЭК России от 18.02.2013 № 21.
  4. Иными нормативными правовыми актами в сфере персональных данных, действующими на территории РФ.

2 Состав обрабатываемых данных
2.1. Данные, получаемые от Пользователя
  1. Идентификационные данные Telegram: уникальный числовой идентификатор (Telegram ID), имя пользователя (username), отображаемое имя — передаются автоматически при взаимодействии с Telegram-ботом.
  2. Токен МЭШ — временный авторизационный ключ, предоставляемый Пользователем самостоятельно и добровольно для получения данных о домашних заданиях.
  3. Пользовательские настройки: параметры решения заданий (время выполнения, количество ошибок), выбранный тарифный план.
  4. История взаимодействия с ИИ-ассистентом: текстовые запросы, направленные Пользователем в рамках сессии.
  5. Данные о подписке: тарифный план, дата начала и окончания подписки, баланс единиц генерации.
  6. Реферальные данные: информация о приглашённых Пользователях и полученных бонусах.
2.2. Данные, которые НЕ собираются
✅ Оператор не собирает и не обрабатывает: пароли от учётных записей МЭШ и Telegram, платёжные реквизиты (обрабатываются исключительно платёжными системами), содержание личной переписки вне интерфейса бота, биометрические данные, специальные категории персональных данных (ст. 10 ФЗ-152).

3.1. Цели обработки персональных данных

  2. Исполнение договора об оказании услуг (Пользовательского соглашения).
  3. Управление подпиской, учёт баланса единиц генерации и реферальных начислений.
  4. Обеспечение контекста диалога с ИИ-ассистентом в рамках сессии.
  5. Обработка обращений в службу поддержки.
  6. Соблюдение требований применимого законодательства РФ.
3.2. Правовые основания обработки
  1. Согласие субъекта персональных данных (ст. 6, ч. 1, п. 1 ФЗ-152).

4 Особые условия обработки токена МЭШ
4.1. Назначение и ограничения использования
⚠️ Токен МЭШ является чувствительными данными и обрабатывается с применением повышенных мер защиты.
  1. Токен МЭШ используется исключительно для получения перечня домашних заданий из учётной записи Пользователя и их автоматического выполнения — и ни для каких иных целей.
  2. Токен не передаётся третьим лицам, за исключением случаев, прямо указанных в настоящей Политике.
  3. Токен хранится в зашифрованном виде.
  4. Пользователь вправе отозвать токен в любой момент через команду /mesh → «Отвязать».
  5. По истечении срока действия токен автоматически удаляется из системы.
  6. Оператор не несёт ответственности за последствия, возникшие в результате компрометации токена по вине Пользователя.
5 Передача данных третьим лицам
5.1. Общий принцип

Персональные данные Пользователей не продаются, не передаются и не раскрываются третьим лицам в коммерческих целях.

5.2. Допустимая передача данных

Данные могут передаваться следующим третьим лицам исключительно в объёме, необходимом для оказания Сервиса:

  1. Anthropic, Inc. (Claude API) — текстовые запросы Пользователя к ИИ-ассистенту передаются для генерации ответов. Обработка регулируется Политикой конфиденциальности Anthropic (anthropic.com/privacy). Пользователь осведомлён о данной передаче.
  2. МЭШ API — токен авторизации передаётся для получения данных о домашних заданиях Пользователя.
  3. Платёжные системы — при оформлении подписки платёжные данные обрабатываются непосредственно платёжной системой; Оператор получает только подтверждение факта оплаты.
  4. Уполномоченные органы государственной власти РФ — при наличии законного требования в соответствии с применимым законодательством.

7.1. Перечень прав

В соответствии со статьями 14-17 ФЗ-152 Пользователь вправе:

  2. Потребовать уточнения неточных или неполных персональных данных.
  3. Потребовать удаления персональных данных в случаях, предусмотренных ФЗ-152.
  4. Отозвать согласие на обработку персональных данных (влечёт прекращение возможности использования Сервиса).

  3. Удаление данных через команду /sup — в течение 30 дней с момента подтверждённого запроса.
8 Сроки хранения и удаление данных
8.1. Сроки хранения
  1. Персональные данные хранятся в течение всего периода использования Сервиса Пользователем.
  2. После прекращения использования данные хранятся не более 12 (двенадцати) месяцев, после чего удаляются автоматически.
  3. История диалога с ИИ-ассистентом удаляется по команде /reset или автоматически при каждой новой сессии.
  4. Токен МЭШ хранится до его отзыва Пользователем или истечения срока действия.
8.2. Удаление данных по запросу
  1. По запросу Пользователя все его персональные данные удаляются в течение 30 (тридцати) календарных дней.
