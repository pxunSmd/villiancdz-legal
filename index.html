Что делает Сервис
2

VillianCDZ — Telegram-бот, который помогает школьникам с домашними заданиями из МЭШ (Московская электронная школа). Функции Сервиса:

  • Авторизация через токен МЭШ и просмотр актуальных ДЗ по неделям и дням
  • Автоматическое решение цифровых домашних заданий (ЦДЗ) с настройкой времени выполнения и допустимого числа ошибок
  • ИИ-ассистент на базе Claude (Anthropic) для помощи в учёбе

Сервис является вспомогательным инструментом. Администратор не гарантирует академических результатов и не поощряет использование бота в ущерб самостоятельному обучению.

О документе

Настоящая Политика конфиденциальности описывает, какие данные собирает VillianCDZ, как они используются и защищаются.

📦
Какие данные мы собираем
2
  • Telegram ID и имя — уникальный идентификатор и имя из вашего Telegram-профиля
  • Токен МЭШ — временный токен авторизации для доступа к вашим заданиям
  • Настройки решения — выбранное время выполнения и допустимое число ошибок
  • История диалога с ИИ — для поддержания контекста беседы (сбрасывается по /reset)
  • Статус подписки — тариф, дата окончания, баланс запросов

Мы не собираем: пароли от МЭШ, платёжные реквизиты (обрабатываются платёжными системами), личную переписку вне бота.

🎯
Цели обработки данных
3

Данные используются исключительно для:

  • Предоставления функций Сервиса (просмотр ДЗ, автоматическое решение)
  • Сохранения ваших настроек между сессиями
  • Поддержания контекста диалога с ИИ-ассистентом
  • Управления подпиской и балансом
  • Технической поддержки при обращении через /sup
🔑
Токен МЭШ — особые условия
4

Токен МЭШ — чувствительные данные. Мы обеспечиваем их особую защиту:

  • Токен используется исключительно для получения списка ваших ДЗ и их выполнения
  • Токен не передаётся третьим лицам ни при каких обстоятельствах
  • Вы можете удалить токен в любой момент через команду /mesh → Отвязать
  • При истечении срока действия токен автоматически удаляется из системы

⚠️ Администратор не несёт ответственности за компрометацию токена, если она произошла по вине пользователя.

  • Государственные органы — исключительно по требованию закона
